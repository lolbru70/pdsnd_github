### Project Title
Explore US Bikeshare Data

### Description
Purpose of this project is to develop Python code to analyse Bikshare data from 3 differents cities.
You can :
> choose the city,
> see time statistics,
> see station statistics,
> see trip statistics,
> see user statistics,
> see raw data
